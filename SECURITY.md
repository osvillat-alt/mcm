# Security Policy

## Supported Versions

Currently, we support the latest version of the website deployed on the `main` branch.

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability, please do NOT create a public issue.

1.  **Email**: Please send details to the repository owner (or your contact email).
2.  **Response**: We will acknowledge your report within 48 hours.
3.  **Correction**: We will fix valid issues as soon as possible and notify you when it is resolved.
